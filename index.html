Sobre

Sou um desenvolvedor web apaixonado por criar experiências interativas e enriquecedoras para usuários. sou desenvolvedor web, tenho habilidades sólidas em HTML, CSS e JavaScript, bem como experiência em trabalhar com frameworks modernos como React e jquery.

Tenho trabalhado em diversos projetos, desde pequenas landing pages até aplicativos web complexos. Um dos meus projetos mais recentes foi o desenvolvimento de um web app de controle de orçamentos e cotas de uma empresa, Posso desenvolver uma página perfeita para você vender o seu trabalho, ou produto que deseja.

Além de minha experiência técnica, também sou apaixonado por design e usabilidade. Acredito que a combinação dessas habilidades me permite criar projetos que são não só funcionais, mas também esteticamente Bonitos de se ver.
